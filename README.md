## 推荐DDD的三个理由

* 业务逻辑可视化
* 灵活的业务拓展性
* 轻量化的业务单元测试

## 领域驱动设计开发流程

* 事件风暴
* 划分子域
* 设计模型
* 单元测试
* 集成业务

## 推荐的框架与软件

### SpringJPA

JPA是ORM框架，可以非常简单的与领域对象相关联。 https://spring.io/projects/spring-data-jpa

### StartUML

StartUML是UML设计软件。https://staruml.io/

### Mural

Mural 是用于画事件风暴图的软件。https://www.mural.co/
